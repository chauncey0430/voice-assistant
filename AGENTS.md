# AGENTS.md

## Coding rules

* Use Python 3.10+
* Prefer simple, modular code
* Keep functions small and readable
* Add basic error handling
* Do not introduce cloud dependencies unless explicitly requested

## Project structure

* core/: audio, stt, tts, router
* skills/: executable actions
* utils/: shared helpers

## Delivery expectations

* Always update README when adding runnable features
* Always keep requirements.txt current
* Prefer minimum viable implementation first
* Summarize changed files at the end
